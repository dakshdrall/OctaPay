# OctaPay - DeFi Financial Super-App

A decentralized finance platform built on Stellar blockchain.

## Live Demo
- Frontend: https://octa-pay-frontend.vercel.app
- Backend API: https://octapay.onrender.com
- Health: https://octapay.onrender.com/health

## Features
- User registration with Stellar testnet wallet auto-creation
- XLM wallet funding via Friendbot
- On-chain transactions on Stellar testnet
- Explorer page showing live on-chain transactions
- Investment and borrowing modules
